Cómo se instala — 2 clics

A · En Claude Code

/plugin marketplace add Danielosongo/origen-editor
/plugin install origen-editor@origen-editor

B · A mano

Descarga la carpeta y ábrela en VS Code o Claude Code de escritorio (arrástrala). Luego doble clic en INSTALAR.bat → “Todo listo ✅”.
